# 🍽️ AI Recipe Generator

An intelligent recipe generation application built using LangChain and Google's Gemini API. Users can enter available ingredients, and the AI generates a complete recipe with a recipe name, ingredients list, and step-by-step cooking instructions.

## Features

- Generate recipes from available ingredients
- AI-powered recipe recommendations
- Simple and interactive user input
- Clean and structured recipe output
- Built using LangChain and Gemini API

## Tech Stack

- Python
- LangChain
- Google Gemini API
- Jupyter Notebook / Google Colab

## How It Works

1. User enters available ingredients.
2. LangChain processes the input through a prompt template.
3. Gemini generates a suitable recipe.
4. The recipe is displayed in a clean and readable format.

## Example Input

banana, honey, bread

## Example Output

### Honey Banana Toast

#### Ingredients
- Bread
- Banana
- Honey

#### Procedure
1. Toast the bread until golden brown.
2. Slice the banana and place it on the toast.
3. Drizzle honey over the banana slices.
4. Serve immediately.
